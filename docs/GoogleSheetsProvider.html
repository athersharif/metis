driveApiUrl

Constructs the URL for Google Drive API. Uses the Google Sheets ID and API key from the REACT_APP_GOOGLE_SHEETS_DOC_ID and REACT_APP_GOOGLE_SHEETS_API_KEY environment variables respectively. The environment variables should be declared as per the Create React App guidelines.

fetchData

Fetches Google Sheets data by first checking when it was modified. If it was modified before the last check then pulls the data from localStorage otherwise makes a network call to Google Sheets API. Helpful when processing large amount of Sheets data.
